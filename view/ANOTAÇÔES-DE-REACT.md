## Oque é o react

- O react é uma biblioteca usada para criar interfaces (Front End)

## Conceito de componentes

- No react você cria interfaces separando itens da tela em "componentes" que pode ser interpretados como uma peça de lego, que quando juntas formam projetos maiores

## State/Estado no React

- State é um objeto que armazena dados que podem mudar ao longo do tempo e afetam a renderização de um componente

- A ferramenta usada para gerenciamento de State se chama "useState" 

## KEY e Renderização de Listas

- Para renderizar uma lista/array, utilizamos a funções de array (como map, filter, find...) para percorrer a lista, podendo assim acessar cada elemento do array e com isso renderizar algo na tela ou nao

- Quando você renderiza normalmente uma propriedade de uma lista o React exige uma key unica para cada elemento da lista para expecificar cada elemento e assim garantir uma atualização eficiente de listas
